Локальный YML

Для запуска проекта в IntelliJ Idea c другими параметрами (например для локальной БД) можно использовать следующий способ:

Рядом с фалом application.yml создаем файл application-local.yml

Сначала будет прочитан первый файл, а потом второй.

Совпадающие значения будут перезаписаны.

Поэтому в локальном можно писать только, то что надо изменить./p>

  • Заходим в Run/Dedug Configurations (Run => Edit Configurations...)
  • Закладка Configuration
  • Поле Environment variables пишем SPRING_PROFILES_ACTIVE=local
  • Сохраняем, запускаем, радуемся

Не забываем поставить application-local.yml в .gitignore
